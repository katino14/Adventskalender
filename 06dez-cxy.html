6. Dezember

Es gibt unzählige Tools, um

interaktive Lerneinheiten

zu erstellen. Die wohl vielfältigste und sich stetig weiterentwickelnde ist H5P.

H5P ist eine freie und quelloffene Software. Die Idee für diese Anwendung stammt von der NDLA, einer staatlich geförderten Organisation aus Norwegen. Ziel war es eine OER-Anwendung zu erfinden. Das Unternehmen Joubel ist seit 2013 die treibende Kraft dahinter.

Wo und wie kann H5P genutzt werden?

1. Auf H5p.org kann man sich mit einer E-Mail registrieren, um Inhalte zu erstellen. Diese Seite dient aber dazu H5P zu probieren und dann die Software, deren Code offen zur Verfügung steht auf eigenen Systemen zu installieren. Daher garantieren die Betreiber nicht, dass alle erstellten Inhalte dauerhaft gespeichert werden. Auf den ersten Blick abschreckend ist die englische Sprache.
2. Nele Hirsch vom eBildungslabor bietet auf EinstiegH5p eine deutsche Variante an, um H5P offen und unkompliziert auszuprobieren. Dabei ist keine Registrierung und auch kein Passwort erforderlich. Allerdings werden die Test-Inhalte nur für sechs Stunden gespeichert und danach gelöscht.
3. Auf apps.zum.de können registrierte Nutzer H5P Inhalte erstellen. Die Oberfläche ist auch hier deutsch. Die erstellten Inhalte können dann über einen Link abgerufen werden oder auf ZUM-Seiten eingebunden werden. Die Server stehen dabei in Deutschland.

Derzeit kann man auf H5P über 45 unterschiedliche Inhaltstypen erstellen. Nach einer Einarbeitungsphase gelingen so vielfältige Übungen. Beispiele, die mit H5P erstellt werden können:

Lückentext (Drag the Words)
Zur Selbstkontrolle bei Übungsaufgaben (Course Presentation)
Informationen zu DUBAS (Image Hotspots)
